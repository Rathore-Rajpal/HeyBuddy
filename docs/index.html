Get Started Today

Download and set up Buddy in minutes

Buddy v1.0.0

Windows 10/11 • Python 3.8+

git clone https://github.com/Rathore-Rajpal/HeyBuddy.git
Download ZIP View on GitHub

Quick Setup:

  1. Run setup.bat to install dependencies
  2. Copy .env.example to .env and add your API keys
  3. Run setup_face_auth.bat to register your face
  4. Launch with LAUNCH_BUDDY.bat
